  1. DOES HOSTING THE OLYMPICS HAVE A CORRELATION TO ACHIEVING EITHER GOLD, SILVER OR BRONZE FROM 2006-2022?

In this Python code snippet, we begin by importing Pandas and Matplotlib libraries, which are essential for data manipulation and visualization, respectively. The data is then loaded from a CSV file named olympic_games.csv into a Pandas DataFrame for processing. The code specifically selects a subset of this data, focusing on rows 0 to 489 and a set of relevant columns, to analyze the medal counts from the Olympic Games.

To distinguish between host and non-host countries, a new column, 'IsHost', is created by comparing the 'Country' field to the 'HostCountry' field within the DataFrame. This boolean column is crucial for the subsequent grouping operation, where the DataFrame is grouped by the 'IsHost' status. For each group, the sum of 'Gold', 'Silver', and 'Bronze' medals is calculated to get the total medal counts for host and non-host countries.

To prepare the data for visualization, the aggregated medal counts DataFrame is transposed,so that each row represents a medal type and each column represents the host status: making it ready for plotting. The code then proceeds to plot a bar chart, with the x-axis representing the type of medal and the y-axis showing the total count of medals. The chart includes a title, axis labels, and a legend that distinguishes between host and non-host countries, providing a clear visual comparison of medal counts across the specified years from 2006 to 2022. The use of plt.show() at the end of the snippet ensures that the generated plot is displayed to the user.

In[34]:
import pandas as pd
import matplotlib.pyplot as plt
data = pd.read_csv("olympic_games.csv")
data_16yr = data.iloc[0:489, [0,1,2,4,5,7,8,9,10] ]
# Filter and determine host status
data_16yr['IsHost'] = data_16yr['country'] == data_16yr['host_country']
# Aggregate medal counts by host status
medal_counts = data_16yr.groupby('IsHost').agg({'gold': 'sum', 'silver': 'sum', 'bronze': 'sum'})
print(medal_counts)
# Transpose for easier plotting
medal_counts = medal_counts.T

# Plotting
medal_counts.plot(kind='bar', figsize=(10, 6))
plt.title('Total Olympic Medal Counts for Host vs Non-Host Countries (2006-2022)')
plt.xlabel('Medal Type')
plt.ylabel('Total Count')
plt.xticks(rotation=0)
plt.legend(title='Host Country', labels=['No', 'Yes'])
